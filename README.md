# angular-countdown
Simple countdown control for AngularJs

## Live demo

[Codepen: Countdown demo](http://codepen.io/HugeHugh/pen/BLvwzr)

## How to Use

``` html
<countdown countdown-id="cd1" countdown-title="Days Until Christmas" target-date="vm.xmas"></countdown>
```
